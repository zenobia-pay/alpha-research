**Suggested UI/Output Changes**
- Show a compact stepper: resolve, create record, upload, deploy, done.
- Add a progress bar or percentage for large uploads, plus an ETA if available.
- If deployment is a separate phase, make that explicit and give it its own status.
- Add a heartbeat like “still uploading” so long waits do not look frozen.
- If the run outlives the terminal session, surface a run id and the next command the user should run.
