# dataset recommendation inventory includes ranked shortlist for the topic

## Product Use

A user asks whether to use an existing dataset or build a new one for housing affordability.

## Why This Test

This protects dataset-choice UX after the journey repairs. The answer should recommend from real datasets and avoid leading with a build plan when a usable match exists.

## Actions Taken

The fake client returns three remote datasets with one housing-specific ready environment.

## Assertions Made

The response starts with the primary dataset, explains why the housing dataset wins, says it is ready, and asks for the geography needed to finalize use.
