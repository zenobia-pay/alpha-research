# dataset recommendation inventory includes ranked shortlist for the topic

## Product Use

A user asks for a dataset recommendation around a research topic. The product should show a ranked shortlist so the user can see the chosen dataset and plausible alternatives.

## Why This Test

Recommendation transparency matters when multiple datasets could fit. A shortlist helps users understand why one dataset was preferred and what tradeoffs remain.

## Actions Taken

The harness returns several remote datasets with differing topic fit and readiness.

## Assertions Made

The response acknowledges the catalog, recommends the top dataset, explains why it ranks first, and asks a focused scoping question.
