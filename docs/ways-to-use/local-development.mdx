---
title: "Local Development"
description: "Run Alpha Research locally from the alpha-datasets repository."
---

Install dependencies:

```bash
npm install
```

Run the full local stack:

```bash
npm run dev:stack
```

Local services:

| Service | URL |
| --- | --- |
| API | `http://localhost:8787` |
| Frontend | `http://localhost:4173` |

Fixture datasets:

- `demo-tweets`
- `demo-econ`

Quality gates:

```bash
npm run build
npm run typecheck
npm run test
npm run harness:check
npm run docs:check
```
