description: "Start research from canonical public datasets."

Use public datasets when your question fits an existing canonical environment.

Good prompts:

```bash
research --prompt "Which public dataset should I use for housing affordability?"
research --prompt "Use econ to inspect county-month housing permit and income coverage."
research --prompt "Describe the econ dataset and its source inventory."
```

Start with [Public Datasets](/public-datasets) to understand what each canonical environment covers.
