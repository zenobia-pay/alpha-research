---
title: "Run Research Jobs"
description: "Launch remote analysis, transformation, labeling, and documentation jobs."
---

Remote runs are useful when work needs cloud resources, dataset mounts, or durable artifacts.

Common run types:

- dataset description and profiling
- data transformation
- labeling
- aggregate queries
- hypothesis analysis
- source discovery and expansion planning
- provenance auditing

When a run starts, the CLI should return immediately with a run id and dashboard link unless you explicitly ask it to wait.
