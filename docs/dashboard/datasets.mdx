---
title: "Datasets in the Dashboard"
description: "Inspect public and private datasets from the dashboard."
---

The datasets view separates private workspace datasets from public canonical datasets.

Use it to:

- compare dataset purpose, coverage, freshness, and limitations
- open dataset details
- review generated briefings and profile evidence
- start a dataset describe run when documentation needs to be refreshed
- move from a dataset to its related runs

Public dataset documentation starts at [Public Datasets](/public-datasets).
