The RESEARCH CLI is the fastest way to work with Alpha Research from a terminal. It can list datasets, inspect dataset profiles, start remote runs, poll status, and debug failures.

## Default behavior

- `research` opens the interactive assistant UI.
- `research --prompt "<prompt>"` runs a one-shot prompt.
- `research login` signs in through `alpharesearch.nyc`.
- `research debug run <run-id>` collects remote run evidence before guessing about failures.
