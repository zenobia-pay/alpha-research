---
title: "Common CLI Commands"
description: "Useful RESEARCH CLI commands for day-to-day dataset work."
---

## Start an interactive session

```bash
research
```

## Run a one-shot prompt

```bash
research --prompt "show remote datasets"
```

## Ask for a dataset recommendation

```bash
research --prompt "I want to research housing affordability. Which dataset should I use?"
```

## Start or inspect a public-data environment

```bash
research --prompt "Use the canonical econ dataset to study county-month housing permit changes."
```

## Retrieve recent results

```bash
research --prompt "Show me the latest completed run results."
```

## Run local checks from this repository

```bash
npm run build
npm run typecheck
npm run test
npm run harness:check
```
