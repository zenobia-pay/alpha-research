---
title: "Install the CLI"
description: "Install or run the RESEARCH CLI."
---

Install globally:

```bash
npm install -g @rprend/research
research
```

Run without a global install:

```bash
npx @rprend/research@latest
```

From this repository checkout:

```bash
npm install
npm run build
npm run cli -- --help
```

## Local development stack

```bash
npm run dev:stack
```

The local API runs on `http://localhost:8787` and the local frontend runs on `http://localhost:4173`.
