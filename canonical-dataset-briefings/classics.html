notes
Pleiades period vocabulary spans from ~2.6M BCE to 21st century CE; Trismegistos geo records concentrate between ca. 1539 BCE and 1998 CE. Text corpora focus on works composed before 1000 CE but metadata includes modern publication dates.

Geography

Global ancient world with emphasis on Mediterranean, Near East, and North Africa (Pleiades, Trismegistos) plus modern coordinates for Wikidata places.

Quality And Limitations

Quality

pleiades
README documents frequent updates (multiple times weekly as of 2025-05-28) and details on coordinate accuracy and data validation procedures.
first1k
Repository badges track metadata completeness, text coverage, and automated checks via Perseids Hook integration.
perseus
Catalog MODS/MADS files follow Library of Congress schemas; CSV crosswalk enables linking to PHI/TLG identifiers.
trismegistos
Geo dump generated from Trismegistos Data Services export interface; no explicit QA metadata provided.
wikidata
Snapshots inherit Wikidata's community curation; JSON includes SPARQL head/results structure.

Limitations

value
`dataset/wikidata/classical_authors.json` includes control characters that break strict JSON parsing.
`dataset/wikidata/classical_literary_works.json` currently contains an empty result set.
`dataset/trismegistos/trismegistos_geo_places.csv` has many rows with blank `tm_geo_id` values and mixed BC/AD date strings requiring normalization.
Pleiades time period bounds mix BC/AD labels with textual numerals; conversion to numeric ranges needs explicit parsing.
Perseus catalog metadata includes rows with placeholder `urn` values (e.g., `none`) and requires deduplication when joining to MODS XML.
